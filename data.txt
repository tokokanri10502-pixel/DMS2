### 第1章：ITインフラ・基礎知識編（デジタルの土台）

**Q1. Webサイトの通信を暗号化し、第三者による盗聴を防ぐ仕組み（URLがhttpsで始まるもの）は？**

1. SEO対策
2. SSL/TLS通信
3. DNSルーティング
4. FTP転送
【正解】2
**【解説】** 個人情報やパスワードを安全に送受信するための必須技術です。

**Q2. 複数のシステムを、1つのIDとパスワードでログインして利用できる仕組みは？**

1. VPN
2. 二段階認証
3. エンドツーエンド暗号化
4. シングルサインオン（SSO）
【正解】4
**【解説】** 利便性とセキュリティを両立させる仕組みで、社内システムなどで導入が進んでいます。

**Q3. スマホやPCなど、端末の画面サイズに合わせてレイアウトが自動で最適化される手法は？**

1. レスポンシブデザイン
2. パララックスデザイン
3. フラットデザイン
4. アクセシビリティ
【正解】1
**【解説】** デバイスごとに別々のサイトを作らず、一つのサイトで表示を切り替えます。

**Q4. インターネット上の機器を識別するために割り当てられる「ネットワーク上の住所」は？**

1. MACアドレス
2. IPアドレス
3. ドメイン名
4. サブネットマスク
【正解】2
**【解説】** 通信を行う全ての機器が持つ一意の番号です。

**Q5. 次回訪問時の表示を高速化するために、ブラウザがデータを一時的に保存する仕組みは？**

1. Cookie
2. アーカイブ
3. キャッシュ
4. バックアップ
【正解】3
**【解説】** 2回目以降の表示を早めますが、古い情報が残ってしまう原因にもなります。

**Q6. テレワーク先などから安全に社内システムへ接続するための仮想専用線技術は？**

1. VPN
2. FTP
3. DNS
4. LAN
【正解】1
**【解説】** 公衆回線の中に専用の「トンネル」を作るイメージで通信を保護します。

**Q7. パソコンやスマホの土台となる「基本ソフトウェア（WindowsやiOSなど）」の総称は？**

1. ブラウザ
2. ミドルウェア
3. アンチウイルス
4. OS（オペレーティングシステム）
【正解】4
**【解説】** あらゆるアプリケーションを動かすための最も基礎となるソフトです。

**Q8. ブラウザ等に後から便利な機能を追加するための小さなプログラムの総称は？**

1. ドライバ
2. パッチ
3. プラグイン（拡張機能）
4. ファームウェア
【正解】3
**【解説】** 自分の使い勝手に合わせてツールをカスタマイズできます。

**Q9. ネット経由でソフトウェアの機能を月額課金などで利用する形態（Zoom等）は？**

1. SaaS（サース）
2. オンプレミス
3. ローカルホスト
4. イントラネット
【正解】1
**【解説】** ソフトウェアを「購入」するのではなく「利用」する、現在の主流です。

**Q10. ウイルスなど、デバイスに悪意のある動作をさせるソフトウェアの総称は？**

1. スパム
2. ファイアウォール
3. トラッキング
4. マルウェア
【正解】4
**【解説】** コンピュータウイルスやスパイウェアなどの総称です。
